## 🚀 Запуск проекта

Проект не требует сборки – это статическое веб-приложение.

1. **Скачайте или клонируйте репозиторий**
2. **Откройте файл `index.html` в браузере**
3. Используйте любой современный браузер (Chrome, Firefox, Edge, Safari)
4. Для корректной работы WebRTC и микрофона/камеры откройте страницу по **HTTPS** (или используйте `localhost`)

### Настройка бэкенда
Проект использует внешний облачный сервис (аутентификация, база данных).  
При необходимости замените конфигурационные данные в начале скрипта на свои.

## 📱 Адаптивность

- **Мобильные устройства (до 640px)** – чат на весь экран, боковое меню выезжает свайпом или кнопкой
- **Планшеты (641–1000px)** – двухпанельный режим, но с возможностью сворачивания списка чатов
- **ПК (>1000px)** – классический двухпанельный интерфейс

### 1. Общие положения
- Правила обязательны для всех пользователей NexLink.
- Регистрируясь в приложении, вы соглашаетесь с настоящими Правилами.

- Пользователь обязан уважать других участников сервиса.

### 2. Регистрация и аккаунт
- Запрещено создавать более одного аккаунта (мультиаккаунтинг).
- Запрещено регистрироваться с использованием чужих данных.
- Запрещено использовать оскорбительные, нецензурные или провокационные юзернеймы и имена.
- Необходимо указывать реальное имя и фамилию в профиле (рекомендуется).
- Запрещено передавать свой аккаунт третьим лицам.
- Запрещено использовать аккаунт для совершения противоправных действий.

### 3. Поведение и общение
- Запрещено оскорблять, унижать и проявлять агрессию по отношению к другим пользователям.
- Запрещено использовать нецензурную лексику в публичных чатах и группах.
- Запрещен спам, флуд и повторяющиеся одинаковые сообщения.
- Запрещено провоцировать конфликты (троллинг).
- Запрещено распространять слухи и клевету.
- Требуется соблюдать культуру общения и этикет.
- Запрещено преследовать (stalking) других пользователей.

### 4. Запрещённый контент
- Запрещен порнографический, эротический и сексуально откровенный контент.
- Запрещен контент, пропагандирующий насилие, жестокость, суицид.
- Запрещена пропаганда наркотиков, алкоголя и психоактивных веществ.
- Запрещен экстремистский, террористический и националистический контент.
- Запрещено размещать материалы, нарушающие авторские права.
- Запрещено размещать личные данные других людей без их согласия.
- Запрещен контент 18+ в общедоступных чатах и каналах.

> ⚠️ **Важно:** Администрация оставляет за собой право удалять любой контент, нарушающий данные правила, без предварительного уведомления.

### 5. Чаты, группы и каналы
- Запрещено создавать чаты/группы с целью нарушения правил.
- Администраторы групп обязаны следить за соблюдением правил внутри своего сообщества.
- Запрещено массово приглашать пользователей в чаты без их согласия.
- В каналах запрещено размещать рекламу без разрешения владельца.
- Запрещено выдавать себя за администрацию NexLink.

### 6. Боты и автоматизация
- Создавать ботов можно только через официальную функцию приложения.
- Запрещено использовать внешние скрипты и ботов для спама.
- Боты не должны нарушать общие правила сервиса.
- Запрещено создавать ботов, имитирующих реальных людей.

### 7. Звонки и голосовое общение
- Запрещено совершать звонки с целью домогательства.
- Во время звонка запрещено включать громкую музыку или шум без согласия собеседника.
- Запрещено записывать звонки и распространять их без согласия всех участников.
- Во время видеозвонков пользователь должен соблюдать приличия (одежда, фон).

### 8. Конфиденциальность и безопасность
- Запрещено просить у других пользователей пароли, коды и финансовые данные.
- Рекомендуется не распространять свои личные данные.
- Запрещено публиковать чужие переписки без согласия.
- Пользователь несёт ответственность за сохранность своего аккаунта.

### 9. Реклама и коммерческая деятельность
- Запрещена реклама без согласования с администрацией.
- Запрещено заниматься мошенничеством и финансовыми пирамидами.
- Разрешена только тематическая реклама в специально созданных каналах.

### 10. Модерация и наказания
- За нарушение правил предусмотрены предупреждение, мут, бан аккаунта.
- Срок бана определяется тяжестью нарушения.

- Решение администрации является окончательным.
- Пользователь может обжаловать бан через поддержку.

### 11. Верификация
