# 💬 NexLink — современный веб-мессенджер

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)
[![Live Demo](https://img.shields.io/badge/demo-nexlink.zapto.org-blue)](https://nexlink.zapto.org/)
[![Made with Firebase](https://img.shields.io/badge/Firebase-039BE5?logo=firebase&logoColor=white)](https://firebase.google.com/)
[![WebRTC](https://img.shields.io/badge/WebRTC-333333?logo=webrtc&logoColor=white)](https://webrtc.org/)

**NexLink** — это полнофункциональный мессенджер, работающий прямо в браузере.  
Никаких установок, только регистрация — и вы уже в сети. Общайтесь в реальном времени, делитесь медиа, звоните, создавайте группы и каналы, настраивайте внешний вид и пользуйтесь встроенными мини‑приложениями — всё в одном месте.

🔗 **Живой сайт:** [https://nexlink.zapto.org/](https://nexlink.zapto.org/)

---

## 🚀 Основные возможности

| Категория | Что умеет |
|-----------|-----------|
| **Аккаунт** | Регистрация по email или через Google, верификация почты, смена пароля, удаление профиля. |
| **Общение** | Личные сообщения, групповые чаты, публичные каналы. Отправка текста, изображений, стикеров и голосовых сообщений. |
| **Звонки** | Аудио‑ и видеозвонки прямо в чате (WebRTC). |
| **Управление сообщениями** | Реакции (❤️😂😮…), ответы, пересылка, редактирование и удаление. |
| **Организация** | Папки для чатов, архивация диалогов, чёрный список. |
| **Приватность** | Гибкие настройки: кто видит ваш статус, почту, кто может приглашать в группы. |
| **Родительский контроль** | Детские аккаунты (до 14 лет) с ограничениями и возможностью запроса на изменение возраста. |
| **Оформление** | 10 встроенных тем + магазин эксклюзивных тем за внутриигровую валюту **Линкоины**. |
| **Мини‑приложения** | Погода, заметки, таймер, калькулятор, календарь, переводчик на 15 языков. |
| **Язык интерфейса** | Русский, английский, арабский. |
| **Безопасность** | Просмотр активных сессий, завершение удалённых устройств, E2EE‑шифрование для приватных чатов. |

## 🧩 Встроенные мини‑приложения

- **Погода** — автоматическое определение геолокации или ручной ввод города.  
- **Заметки** — быстрые записи, хранятся в браузере.  
- **Таймер** — обратный отсчёт с уведомлением.  
- **Калькулятор** — простые арифметические операции.  
- **Календарь** — просмотр месяцев и дней с выделением текущей даты.  
- **Переводчик** — перевод текста на 15 языков (включая русский, английский, арабский, испанский, китайский и др.).

## 🛠️ Как начать использовать

1. Перейдите на сайт [nexlink.zapto.org](https://nexlink.zapto.org/).  
2. Зарегистрируйтесь (или войдите через Google).  
3. Начните общение:
   - Нажмите **✏️** в списке чатов, чтобы найти собеседника.  
   - Откройте чат и используйте **📞** или **📹** для звонка.  
   - Зажмите **🎙️** для записи голосового сообщения.  
   - Долгое нажатие на сообщении (или правый клик на ПК) открывает контекстное меню с дополнительными действиями.

---

## 🚧 Запуск локально (для разработчиков)

1. Скачайте или клонируйте репозиторий.  
2. Откройте файл `index.html` в браузере (рекомендуется использовать локальный сервер, например, **Live Server**).  
3. Настройка Firebase не требуется — приложение уже интегрировано с облачной инфраструктурой.  
4. Для работы сервис‑воркера (`sw.js`) разместите файлы на любом статическом хостинге.

---

## 🧰 Используемые технологии

- **Frontend:** Vanilla JS, HTML5, CSS3  
- **Real‑time:** Firebase Realtime Database / Firestore  
- **Аутентификация:** Firebase Auth (email + Google)  
- **Звонки:** WebRTC  
- **Хостинг:** статический (или Firebase Hosting)  
- **Service Worker:** для кэширования и офлайн‑режима
