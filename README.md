# ✨ [NexLink] — Мессенджер будущего для сообществ  
### Живые комнаты · Анимированные голосовые · 9 тем · Open Source · Self‑Hosted

> **Никакой слежки, рекламы и лишних переписок. Только ваше сообщество и свобода.**

---

## 🧠 Что это?

**NexLink** — это не очередной мессенджер.  
Это **цифровой дом** для вашего комьюнити. Никаких бесконечных личных диалогов — только **живые комнаты**, где рождаются идеи.

Вместо «чата с другом» → **комнаты по интересам**:  
📚 Книжный клуб · 🚀 Стартап · 📸 Фотошедевры · 💻 Dev-команда

---

## ✨ Возможности, которые влюбляют

### 🏠 Комнаты — всё общение здесь
- 🔓 **Публичные** — открыты для всех  
- 🔒 **Приватные** — по ссылке или приглашению  
- 📜 История сообщений сохраняется полностью  
- 👥 Видно, кто онлайн (опционально)

### 🎤 Голосовые с **живой анимацией**
Вместо скучного плеера — **пульсирующий звуковой пузырёк**:
- 15 цветных полосок в такт голосу  
- Плавный прогресс‑бар  
- Круглая кнопка с паузой  
- Формат `m:ss` внутри сообщения  

Это красиво настолько, что хочется отправлять голосовые снова и снова.

### 🌈 9 тем оформления — под любое настроение

| Тема | Настроение |
|------|------------|
| 🌿 Классическая | Лавандовый спокойный стиль |
| 🌊 Океан | Свежесть синих глубин |

*Темы меняют фон, цвета сообщений, панели и кнопки — сохраняются при следующем входе.*

### 🔍 Мгновенный поиск
Начинаете печатать имя или название — результат сразу. Фильтр: только люди / только чаты.

### 🔄 Умное обновление
Кнопка **«🔄 Обновить данные»** подгружает новые комнаты и участников **без перезагрузки страницы**.

### 📎 Файлы и картинки
Скриншоты, документы, мемы — всё поддерживается.

### ⚡ Реальное время
Сообщения появляются мгновенно. Никаких задержек.

### 🔓 Регистрация без данных
Просто **имя + пароль**. Ни телефона, ни email, ни паспорта.

### 🖥️ Работает в браузере
Никаких установок. Открыли ссылку — и вы в игре. С телефона, планшета, ПК.
